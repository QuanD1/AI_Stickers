AI 即時文字轉貼圖輸入法系統

Android IME × NLP × AI Image Generation

葉慧全｜資訊工程

專案簡介

開發一套能把使用者輸入的文字即時轉換成貼圖的系統，結合自然語言處理、圖像生成、影像分割，並做成 Android 輸入法，讓貼圖能直接用在通訊軟體。

技術重點

語言模型：把「文字 + 角色 + 情緒」轉成圖像生成 prompt 圖像生成模型：產生 Q 版風格貼圖 影像分割模型：自動去背，生成透明背景貼圖 文字排版演算法：自動調整字體方向、大小與位置 前端：Flutter（APP 主頁功能）、Kotlin（Android 輸入法介面功能） 後端：Python，部署於 Google Colab，透過 Cloudflare Tunnel 提供外網 API

專案成果

完成「一句話生成一張貼圖」的即時系統 Android 輸入法整合，能在聊天中直接傳貼圖 提升貼圖多樣性與即時互動體驗
