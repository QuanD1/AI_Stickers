Android IME × NLP × AI Image Generation

葉慧全｜資訊工程

專案簡介

本專案建構一套完整之 AI 貼圖生成 Pipeline， 整合 NLP 語意解析、圖像生成模型與影像分割處理。

透過 Android IME 實現於通訊場景中的即時貼圖生成與傳送功能。

技術重點

  • 語言模型：解析文字語意（角色 / 情緒），自動生成圖像 Prompt
  • 圖像生成模型：產生 Q 版風格貼圖
  • 影像分割模型：自動去背，輸出透明背景 PNG
  • 文字排版演算法：自動調整字體大小、方向與位置
  • 前端：Flutter（APP 主頁）、Kotlin（Android IME）
  • 後端：Python，部署於 Google Colab，透過 Cloudflare Tunnel 提供 API

系統架構

系統處理流程為： 使用者輸入 → NLP 解析 → Prompt 生成 → 圖像生成 → 影像分割 → 文字排版 → Android IME 傳送貼圖。

專案成果

  • 實現「一句話生成一張貼圖」之即時 AI 系統
  • 完成 Android IME 整合，可於通訊軟體中直接生成並傳送貼圖
  • 建立完整 NLP → 圖像生成 → 去背 → 排版 之處理 Pipeline
